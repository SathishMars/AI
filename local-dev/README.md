# Local Development Setup

**Run Workflows Next.js app with Docker Nginx Reverse Proxy (Recommended):**
- ✅ Docker Nginx reverse proxy for production-like environment
- ✅ Browser-trusted SSL certificates with mkcert
- ✅ HTTP/2 support for better performance
- ✅ Next.js local dev server with hot reload
- ✅ MongoDB 8.0 Docker container
- ✅ No need to edit `/etc/hosts` (uses localhost)

## 🚀 Quick Start: One Command Setup

### Prerequisites (One-time)

1. **Docker Desktop** - Already have it? Skip to step 2
   - Download from [docker.com](https://www.docker.com/products/docker-desktop)
   - Ensure it's running

2. **Run Setup Script**
   ```bash
   npm run setup
   ```
   
   This command will automatically:
   - ✅ Check Docker is running
   - ✅ Generate browser-trusted SSL certificates for `localhost` using mkcert (in Docker)
   - ✅ Store certificates in `local-dev/nginx/certs/`
   - ✅ Install CA certificate in your system keychain (macOS/Windows/Linux)
   - ✅ No `/etc/hosts` modification needed!

That's it! No local tool installations needed.

### Start Development Environment

Open multiple terminals:

**Terminal 1: Start Nginx Reverse Proxy**
```bash
npm run nginx
```

This command:
- ✅ Checks Docker is running
- ✅ Auto-generates SSL certificates if needed
- ✅ Builds and starts the Nginx container on port 8443
- ✅ Routes `/aime/*` to your local Next.js dev server
- ✅ Routes everything else to `https://testing.app.groupize.com/`
- ✅ Supports HTTP/2 for optimal performance

**Terminal 2: Start Next.js Dev Server**
```bash
npm run dev
```

**Terminal 3 (optional): Monitor Nginx Logs**
```bash
npm run nginx logs
```

**Terminal 4 (optional): Start MongoDB**
```bash
npm run mongodb up
```

### Access Your Application

- **Next.js App**: `https://localhost:8443/aime`
- **Testing Environment**: `https://testing.app.groupize.com/`
- **Health Check**: `https://localhost:8443/health`

### Useful Commands

```bash
npm run nginx help         # Show all available commands
npm run nginx status       # Check container status
npm run nginx restart      # Restart the container
npm run nginx stop         # Stop the container
npm run nginx health       # Health check
npm run nginx logs         # View container logs

npm run mongodb up         # Start MongoDB
npm run mongodb down       # Stop MongoDB
npm run mongodb status     # Check MongoDB status
npm run mongodb health     # Health check
npm run mongodb logs       # View MongoDB logs
npm run mongodb info       # Connection details
```

### About Browser Certificate

With browser-trusted mkcert certificates, you **should NOT see any warnings**:
- ✅ Chrome: Shows green lock icon
- ✅ Firefox: Shows green lock icon  
- ✅ Safari: Shows green lock icon

If you DO see a warning:
```bash
npm run setup              # Re-run setup to regenerate certs
npm run nginx restart      # Restart the container
```

## 🔧 Troubleshooting

### Docker not running
```bash
open -a Docker  # Opens Docker Desktop on macOS
```

### Port already in use
```bash
# Check what's using the ports
lsof -i :8443
lsof -i :80
lsof -i :27017

# Stop Nginx container
npm run nginx stop
```

### Cannot access https://localhost:8443
1. Verify Nginx is running: `npm run nginx status`
2. Check browser console for errors
3. Try health check: `curl -k https://localhost:8443/health`

### Next.js dev server not responding
```bash
# Ensure it's running on port 3000
npm run dev

# Check connectivity
curl -v http://localhost:3000/aime
```

### SSL Certificate issues
```bash
# Delete and regenerate certificates
rm local-dev/nginx/certs/localhost*
npm run setup  # Re-run setup
npm run nginx restart
```

### MongoDB connection issues
```bash
# Check MongoDB status
npm run mongodb status
npm run mongodb health

# View logs
npm run mongodb logs

# Restart MongoDB
npm run mongodb down
npm run mongodb up
```

## 🐛 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| "Docker is not running" | Run `open -a Docker` on macOS |
| "Port 8443 already in use" | Run `lsof -i :8443` and kill the process |
| "SSL certificate warning" | Run `npm run setup` again and restart browser |
| "Cannot connect to localhost:8443" | Check Nginx is running: `npm run nginx status` |
| "MongoDB connection refused" | Run `npm run mongodb up` |
| "Next.js not hot-reloading" | Ensure it's running on port 3000: `npm run dev` |
